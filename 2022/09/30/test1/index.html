#最长上升子序列
##题目
给定一个长度为 N 的数列，求数值严格单调递增的子序列的长度最长是多少。

输入格式
第一行包含整数 N。

第二行包含 N 个整数，表示完整序列。

输出格式
输出一个整数，表示最大长度。

数据范围

1≤N≤1000，

−109≤数列中的数≤109

输入样例：
7 3 1 2 1 8 5 6

输出样例：
4

##思路
dp[i]表示以a[i]为末位的数列的最长上升子序列长度，其只与其前面的元素相关，故宜使用动态规划的思想。
##代码
#include
#include
#include
#include

using namespace std;

const int N = 1e5;

int n;
int ans;
int a[N]; 
int dp[N];

int main (){
    cin >> n;
    for(int i = 0; i < n; i++) cin >> a[i];
    
    for(int i = 0 ; i < n; i++){
        dp[i] = 1;
        for(int j = 0; j < i; j++){
            if(a[i]>a[j]){
                dp[i] = max(dp[i],dp[j]+1);
            }
        }
        ans = max(ans,dp[i]);
    }
    
    cout << ans;
    
    return 0;
}
