    • 底线

      语法如下：

      语法说明如下：

      1. 底线是 = 表示一级标题。
      2. 底线是 - 表示二级标题。
      3. 底线符号的数量至少两个。
      4. 这种语法只支持这两种标题。

    • 语法说明如下：

         1.  在行首插入 # 可标记出标题。
   2.  # 的个数表示了标题的等级。
   3.  建议在 # 后加一个空格（一定要5 加）。
   4.  markdown中最多只支持前六级标题。

      使用规范：

      1. 建议使用 # 。Typora中标题快捷键为： ctrl + （标题级数）。
      2. 标题前后各空一行， # 与文本之间也要有一个空格。不要有多余的空格。
      3. 标题的结尾不要有标点符号。
      4. 标题尽量言简意赅。

1.2 段落与换行

  1. 段落与换行

    • 语法与说明如下：

    ​ 效果：

    ​ 我是第一行 我是第二行

    ​ 效果：

    ​ 我是第一行

    ​ 我是第二行

    ​ 效果：

    ​ 我是第一行
    ​ 我是第二行

    • 使用规范：
      1. 当超过 80 个字符后进行换行。
      2. 在一句话结束（ 。 或 ！ 或 ？ ）之后换行。
      3. 当URL较长时换行

  2. 列表

    • 语法说明：

      在 markdown 中支持使用有序列表和无序列表。

      有序列表由 数字序号 + . + 空格 + 列表内容 来标记。

      无序列表由 */+/- + 空格 + 列表内容 来标记（符号不同效果相同）。

      列表之间可以互相嵌套。

    • 使用规范：

      1. 建议使用 - 来标记无序列表， 因为 * 容易跟粗体和斜体混淆， + 不流行。
      2. 如果一个列表中所有的列表项都没有换行，建议使用一个空格。
      3. 如果列表项有换行，则建议给无序列表使用三个空格，给有序列表使用两个空格。
      4. 如果一个列表中的每个列表项都只有一行，建议列表项之间不要有空行。如果列表项中有换行，建议在列表项之间空一行。
      5. 数字、 字符、 列表符号使英文半角句号（.），句号后加空格。

  3. 分隔线

    在markdown中，分割线由三个或以上的 */-/_ 来标记。

    • 语法说明：
      1. 分割线须使用至少三个以上的 */-/_ 来标记。
      2. 行内不能有其他字符。
      3. 可以在标记符中间加上空格。

#最长上升子序列
##题目
给定一个长度为 N 的数列，求数值严格单调递增的子序列的长度最长是多少。

输入格式
第一行包含整数 N。

第二行包含 N 个整数，表示完整序列。

输出格式
输出一个整数，表示最大长度。

数据范围

1≤N≤1000，

−109≤数列中的数≤109

输入样例：
7 3 1 2 1 8 5 6

输出样例：
4

##思路
dp[i]表示以a[i]为末位的数列的最长上升子序列长度，其只与其前面的元素相关，故宜使用动态规划的思想。
##代码
#include
#include
#include
#include

using namespace std;

const int N = 1e5;

int n;
int ans;
int a[N]; 
int dp[N];

int main (){
    cin >> n;
    for(int i = 0; i < n; i++) cin >> a[i];
    
    for(int i = 0 ; i < n; i++){
        dp[i] = 1;
        for(int j = 0; j < i; j++){
            if(a[i]>a[j]){
                dp[i] = max(dp[i],dp[j]+1);
            }
        }
        ans = max(ans,dp[i]);
    }
    
    cout << ans;
    
    return 0;
}
